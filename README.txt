Code to generate pogo pin jigs, for interfacing with circuits and microcontrollers.  See pogo_pins.scad for more details.

Run get_deps.sh to pull dependencies into a linked folder in your home directory.

MIT license.

(Note - if you need something more complete than this "paste pin coordinates and hit print" script I have here, snhobbs has a much more complete-and-polished-looking project at https://github.com/snhobbs/parametric-test-jig .  I haven't tried it, but it LOOKS pretty good.)
